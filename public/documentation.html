MovieAPI

Objective

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Essential Features

request url response
GET /movies Return a list of all movies.
GET /genres Return a list of all genres.
GET /genres/:Name Return information of a specific genre.
GET /movies/:Title Return movie by it's title.
GET /movies/Genre/:Name Return movies by genre.
GET /directors Return info about all directors.
GET /directors/:Name Return a director's information by name.
GET /users Return a list of all users.
GET /users/:Username Return a user's information by username.
POST /users Add a new user.
PUT /users/:Username Allow user to update the user information
POST /users/:Username/movies/:MovieID Allow user to add a movie to the list of favorites.
DELETE /users/:Username/movies/:MovieID Allow user to remove a movie from the list of favorites.
DELETE /users/:Username Delete a Username
